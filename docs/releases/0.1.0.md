# cc-use 0.1.0

> Initial public release

## Summary

`0.1.0` is the first public release of `cc-use`: a small Claude Code launcher for starting Claude Code with third-party Anthropic-compatible providers.

## Highlights

- Initial CLI package structure
- Provider-profile based launch flow
- Isolated `CLAUDE_CONFIG_DIR` model for provider sessions
- npm package setup for the `cc-use` binary
